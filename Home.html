Thyroid Disease classification

Thyroid disease classification is designed to help doctors and patients make informed decisions about which drug to use to treat thyroid diseases.

The system uses a machine learning algorithm to analyze patient data and predict which drug is likely to be the most effective for a particular patient.

To use the system, simply click the "Predict" button in the navigation bar and enter the patient's age, sex, blood pressure, cholesterol level, and sodium to potassium ratio.

The system will then use this information to make a prediction about which drug is likely to be the most effective for the patient.
